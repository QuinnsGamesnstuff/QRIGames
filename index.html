About

This website, once completed, will feature mini-games that are free to play. Right now there are currently three games to play each will receive regular updates as well as the addition of new games. I make the games myself using Construct 3 which is an easy to use game making website. If you’re interested in making your own game I'd highly recommend it for beginners.
